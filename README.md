# 任务分发与回销追踪平台

## 项目简介

这是一个面向企业内部的任务协同平台，通过H5形态嵌入办公平台，实现任务下发、进度追踪、文件回销、数据看板的一体化管理，替代传统Excel表格的分散式任务管理模式。

![任务分发与回销追踪平台](https://p3-doubao-search-sign.byteimg.com/labis/eefa98a3c817fba20b49169294e0ddf4~tplv-be4g95zd3a-image.jpeg?lk3s=feb11e32&x-expires=1787889613&x-signature=57oKQkN0b9oORyXwtr4DAxvQ2ZQ%3D)

## 核心功能

1. **任务事件定义引擎**：支持四种任务模式（指派型、模板回收型、自主填报型、协同型），满足不同业务场景
2. **任务中心模块**：可视化流程编排、批量导入、智能分派、任务追踪看板
3. **回销管理模块**：执行人工作台、一键填报、离线草稿、文件回收与版本管理
4. **统计分析与预警**：部门维度统计、个人统计、三级预警机制

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **UI框架**：Tailwind CSS v3
- **图标库**：Font Awesome 4.7.0
- **图表库**：Chart.js 4.4.8

## 如何使用

1. 直接打开 `index.html` 文件即可访问平台
2. 系统默认登录为张三（产品技术部）
3. 可以通过左侧导航栏切换不同功能模块

## 功能亮点

1. **响应式设计**：支持PC端和移动端访问，适配不同屏幕尺寸
2. **东航主题色**：采用中国东方航空主题色（蓝色 #00227F 和红色 #E50000）
3. **文件版本管理**：支持反复上传/更新文件，保留历史版本
4. **数据可视化**：直观展示任务完成情况、部门排名等数据
5. **多角色支持**：任务下发者和任务接收者两种角色，满足不同使用场景

## 主要页面

1. **工作台**：概览页面，展示待处理任务、已完成任务、待审核任务等统计信息
2. **创建任务**：三步式任务创建流程，支持多种任务类型
3. **任务管理**：管理和追踪所有任务的执行情况
4. **任务回销**：提交任务完成情况和相关文件
5. **数据统计**：查看任务执行情况的统计数据和趋势分析
6. **系统设置**：配置系统参数和个人偏好

## 注意事项

1. 本项目为演示版本，所有数据均为模拟数据
2. 文件上传功能仅为前端演示，实际使用需要后端支持
3. 系统集成功能（如SSO登录、消息推送）需要与办公平台对接
